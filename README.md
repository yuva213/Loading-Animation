# Loading Animation Collection

A set of minimalist, high-performance loading animations created with SVG and CSS. This project showcases three distinct shapes—Circle, Triangle, and Square—featuring a smooth, hypnotic motion that follows the perimeter of each shape.

## ✨ Features

- **Geometric Variety**: Includes three unique shape animations (Circle, Triangle, and Square).
- **Smooth Motion**: Uses cubic-bezier timing functions for organic, fluid transitions.
- **Pure CSS/SVG**: No external libraries or JavaScript required, ensuring maximum performance and lightweight footprint.
- **Dark Mode Optimized**: Designed with a sleek dark aesthetic and vibrant pink accents.
- **Responsive & Scalable**: SVG-based graphics ensure the animations look sharp at any size.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure and SVG paths.
- **CSS3**: Custom properties (variables), Keyframe animations, and Flexbox layout.
- **Meyer Reset**: Ensures consistent rendering across all modern browsers.

## 🚀 Getting Started

To view the animations locally, follow these simple steps:

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.

No build tools or servers are required.

## 🎨 Customization

You can easily modify the appearance by changing the CSS variables in `style.css`:

```css
.loader {
    --path: white;    /* Color of the shape outline */
    --dot: #f40af0;   /* Color of the moving dot */
    --duration: 3s;   /* Speed of the animation */
}
```
